# Login Sheet

A modern, responsive login page with a clean and user-friendly interface.

## Live Demo

Visit the live version here: [https://skmusa786.github.io/Login_Sheet/](https://skmusa786.github.io/Login_Sheet/)

## Features

- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Clean UI** - Modern, minimalist design with smooth animations
- **Social Login Options** - Quick login with Google, Meta, and Apple
- **Email & Password Form** - Standard login form with input validation styling
- **Hover Effects** - Smooth animations on social media buttons
- **Custom Styling** - Beautiful color scheme with Inter font

## Project Structure

```
Login_Sheet/
├── Index.html          # Main HTML file
├── style.css           # Styling and animations
├── google.png          # Google logo
├── meta.png            # Meta/Facebook logo
├── apple-logo.png      # Apple logo
└── README.md           # This file
```

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Responsive styling with flexbox and animations
- **Font** - Inter (Google Fonts)

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The page will display a login form with email, password, and social login options

## File Descriptions

- **Index.html** - Contains the HTML structure with a login form, input fields, and social login buttons
- **style.css** - Includes all styling, animations, hover effects, and responsive design
- **Logo Images** - google.png, meta.png, and apple-logo.png for social login options
